# Иконки кнопок управления

## Созданные иконки:

### button_off.svg
- Красный квадрат с белым квадратом внутри
- Символизирует состояние "Выключено" (STOP)
- Размер: 50x50 пикселей
- Формат: SVG (векторная графика)

### button_on.svg
- Зеленый квадрат с белым треугольником внутри (Play символ)
- Символизирует состояние "Включено" (START/RUN)
- Размер: 50x50 пикселей
- Формат: SVG (векторная графика)

## Использование:

1. Кликните ПКМ по кнопке CoilButton на мнемосхеме
2. В поле "Иконка ON" выберите `button_on.svg`
3. В поле "Иконка OFF" выберите `button_off.svg`
4. Нажмите OK

## Дополнительные иконки:

Вы можете создать свои собственные иконки:
- Рекомендуемый размер: 50x50 или 64x64 пикселей
- Форматы: SVG (предпочтительно), PNG, JPG
- Для PNG используйте прозрачный фон
- Цветовая схема: 
  - OFF: красный (#DC2626)
  - ON: зеленый (#16A34A)
